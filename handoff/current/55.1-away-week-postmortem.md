## 8. Kill-switch audit (06-05) — verdict from arithmetic, not presumption

- **Thresholds from live config (file path cite):** `backend/config/settings.py:453` `paper_daily_loss_limit_pct = 4.0`; `:454` `paper_trailing_dd_limit_pct = 10.0`. (The UI's "daily −1.5%/4%, trail −0.1%/10%" in the operator screenshot are current-reading/limit pairs; the Risk-Monitor card's "-15%" is a separate hardcoded display threshold — inconsistency noted as a finding candidate.)
- **Fields consumed:** `check_and_enforce_kill_switch` (`paper_trader.py:1011-1058`) reads `portfolio["total_nav"]` (`:1019`) — the BQ-stored NAV. §2-§4 show stored NAV was **clean** all week (the FX corruption is trade-ledger + display only) → **the measurement-failure hypothesis is REFUTED**: the switch saw correct data.
- **Audit trail (`handoff/kill_switch_audit.jsonl`):** `sod_snapshot` 06-04 nav=24,541.50; 06-05 nav=23,862.58 (at 19:02 UTC, same instant as evaluation); `peak_update` high-water 24,666.57 (06-03). No `pause` events 06-02..06-10 (the only pauses were two manual pause/resume pairs 06-01 15:47, trigger=manual).
- **Arithmetic:**
  - Daily leg as implemented: SOD is anchored at cycle time (`:1032`), evaluation happens the same instant → daily_loss_pct = (23,862.58−23,860.56)/23,862.58 ≈ **0.0%** < 4.0 → no breach. (Matches the on-screen "0.0%".)
  - Daily leg on true day-over-day NAV: (24,541.50−23,862.58)/24,541.50 = **2.77%** (snapshots: 2.82%) < 4.0 → no breach either.
  - Trailing leg: (24,666.57−23,862.58)/24,666.57 = **3.26%** < 10.0 → no breach. (On-screen "3.4%" ✓.)
- **Verdict: CORRECTLY-DID-NOT-TRIP.** Even the worst honest measure of the day (−2.82%) is below the 4% daily limit, and trailing DD never exceeded 3.4% vs the 10% limit. The digest's "−3.5%" overstated the day.
- **Structural observation (finding candidate, NOT a threshold failure):** because SOD is captured at the once-daily cycle's own evaluation instant, the daily-loss leg compares NAV to itself and is **structurally ≈0 under once-daily cadence** — it can effectively never trip; only the trailing-DD leg has teeth. This is a design limitation to weigh in 55.3, not a 06-05 defect.
